# Development Environments

## Version 
Middleware required for development and supported versions
```
node >= v17.x
```

## Initial Settings
```
git clone -b develop git@github.com:Next2D/player.git
cd player
npm install
```

## Start Development
```
npm start
```

## Unit Test
```
npm test
```

## ESLint
```
npm run lint
```

## Export minify
```
npm run build
```
